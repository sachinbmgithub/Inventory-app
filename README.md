# Inventory Management System

## Features

- Add Item
- Update Item
- Delete Item
- Fetch Items
- MySQL JOIN
- MVC Architecture
- Responsive UI

---

## Backend Setup

```bash
cd backend
npm install
nodemon server.js
```

---

## Frontend Setup

Open frontend/index.html using Live Server.

---

## Database Setup

Import database.sql into MySQL.
